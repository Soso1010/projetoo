Modulo 2 - tabelas e formularios

Nome Definição Sintaxe
table cria uma tabela <table>conteudo</table>
tr cria uma linha <tr>conteudo</tr>
td cria uma celula <td>conteudo</td>
th define uma celula de cabeçalho <th>conteudo</th>
thead criar um cabeçalho de tabela <thead>conteudo</thead>
tbody agrupar o conteúdo do corpo em uma tabela <tbody>conteudo</tbody>
tfoot definir o rodapé de uma tabela <tfoot>conteudo</tfoot>
celulas vazias usar a tr ou a td vazias <tr/>ou <td/>
atributo colspan agrupa colunas e é colocado dentro da tag td <td colspan="numero de colunas">conteudo</td>
atributo rowspan agrupa linhas e é colocado dentro da tag th <th rowspan="numero de linhas">conteudo</th>
caption especifica a legenda ou um titulo de uma tabela <caption>titulo</caption>
colgroup grupo de colunas <colgroup>conteudo</colgroup>
col define as propriedades da coluna para cada coluna especificada dentro da tag colgroup <col atributo="valor">
form é o elemento que formalmente define o formulario e os atributos que definem a maneira como esse formulario se comporta <form>conteudo</form>
metodo GET é o metodo usado pelo navegador para pedir ao servidor para enviar de volta um determinado assunto <form action="link" method= "get">
metodo post é o metodo que o navegador envia ao servidor para pedir uma resposta que leva em conta os dados fornecidos no copo da solicitação HTTP <form action="link" method= "post">
metodo post é o metodo que o navegador envia ao servidor para pedir uma resposta que leva em conta os dados fornecidos no copo da solicitação HTTP <form action="link" method= "post">
input é usado para criar controles interativos para formularios baseados na web para receber dados do usuario <input>
type button cria um botao <input type="button">
type submit cria um botao para enviar o formulario <input type="submit">
type checkbox uma caixa de marcação <input type="button">
type radio um botão de escolha <input type="radio">
type color um controle para especificar cores <input type="color">
type date um controle para inserir uma data <input type="date">
type datetime-local um controle para inserir data e horário <input type="datetime-local">
type month um controle para inserir um número de ponto flutuante <input type="month">
type time um controle para inserir um horário sem fuso horário <input type="time">
type email um campo para editar um endereço de e-mail <input type="email">
type url um campo para editar uma URL <input type="url">
type password um campo de texto com uma só linha cujo valor é obscurecido <input type="password">
type week um controle para inserir uma data consistindo de ano da semana e número da semana sem fuso horário <input type="week">
type search um campo de texto com uma só linha para digitar termos de busca <input type="search">
type hidden um controle que não é exibido mas cujo valor é enviado ao servidor <input type="hidden">
type image define uma imagem como botão de envio do formulario Html <input type="image">
type file um controle que permite ao usuario selecionar um arquivo <input type="file">
type range um controle para inserir um número cujo valor exato não é importante <input type="range">
type number um controle para inserir um número de ponto flutuante <input type="number">
type tel um controle para inserir um número de telefone <input type="tel">
type text um campo de texto com uma só linha <input type="text">
tag textarea um controle de entrada de texto de varias linhas <textarea id="w3review" name="w3review" rows="4" cols="50">
