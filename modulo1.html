Modulo 1 - elementos e estrutura basica html

Nome Definição Sintaxe
comentarios um trecho que serve para explicar algo e nao interfereno código <!-- isso é um comentário -->
espaço em branco coloca um espaço em branco semelhante a apertar uma vez a tecla de espaço &nbsp;
DOCTYPE declaração de tipo de documento diz para o navegador que é um documento html e é a primeira tag <!DOCTYPE html>
elemento html ele indica o inicio e o fim do código <html> ... </html>
cabeçalho elemento principal que agrupa as tags meta e de estilo <head> ... </head>
titulo é o texto que aparece na guia do navegador (tem que vir dentro da head) <title> texto do titulo </title>
body envolve todo o conteudo a ser exibido na pagina <body> ... </body>
elementos de cabeçalho tags h1 a h6 cria titulos e subtitulos <h1> texto </h1>
paragrafo cria um paragrafo <p> texto </p>
quebra de linha cria uma quebra de linha semelhante a apertar o enter <br>
negrito destaca um trecho <b> texto </b>
destaque destaca um trecho <strong> texto </strong>
ênfase italico <em> texto </em>
lista ordenada cria uma lista ordenada <ol> ... </ol>
lista não ordenada cria uma lista não ordenada <ul> ... </ul>
item de lista cria um item de lista <li> conteudo </li>
caminho de arquivo indica um link, local ou nome do arquivo a ser usado ex: url ou arquivo.png
elemento de imagem indica uma imagem que irá exibir no site <img src="">
elemento de video indica um video que irá ser exibido no site <video src="">
elemento ancora linka o documento atual com outro ou uma sessão da própria pagina <a href="">conteudo </a>
atributo alvo (target) mostra onde será aberto o link se vai ser na propria guia ou em outra esse atributo vem dentro da tag a target="_blank" (abre na nova guia) target="_selfie" (abre na mesma pagina)
link para uma parte diferente da pagina conecta para uma sessão da pagina, indicada no atributo href href="#id"
elemento div agrupa elementos verticalmente <div>conteudo </div>
elemento span agrupa elementos horizontalmente <span>conteudo </span>
atributo id especifica uma identificação unica para determinado item id=""
